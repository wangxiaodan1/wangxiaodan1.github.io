    • 创建react、webpack 项目

    一、创建一个react项目

    初始化项目

    • 新建一个文件夹 > 然后在命令行下对项目进行初始化
    • 初始化的步骤
      • 下载node.js，并安装
      • 进入项目目录运行：npm init > 文件夹内多出了一个 package.json 文件
      • 直接用 npm 安装的速度很慢，我们可以用 cnpm install 命令来代替 npm install 命令
        • 安装命令 npm install -g cnpm –registry=https://registry.npm.taobao.org

    二、安装及配置 webpack

    1.webpack是什么？

    Webpack 是一个前端资源加载/打包工具，它能够将各种资源如 js、css、图片作为模块载入并进行打包

    2.webpack的配置

    • 安装 webpack：cnpm install –save-dev webpack(配置的东西放在devDependencies中， 程序中使用的包放在Dependencies)
    • webpack 的测试
      • 在非全局模式安装的webpack的测试， 可以在package.json中的scripts下添加命令 如：”build”: “webpack ./index.js ./build.js” -> 运行 npm run build -> 将 index.js 文件打包成名为 bundle.js 的文件 -> 运行 node build.js -> 打印出index文件中的内容
    • webpack 的配置文件
      • 新建 webpack.config.js 文件：
    var path = require('path');
var webpack = require('webpack');

module.exports = {
 // webpack 进行打包的入口文件，这里 webpack 从根目录下的 index.js 开始进行打包
 entry: [
   './index'
 ],
 // webpack 打包后的输出文件的路径
 output: {
   path: path.join(__dirname, 'dist'), // 文件放至当前路径下的 dist 文件夹
   filename: 'bundle.js',  // 将打包后的输出文件命名为 bundle.js
 }
}

    html 文件的引用路径相应地改成：

    <script src="./dist/bundle.js"></script>

    三、关于babel

    Babel 是什么？

    因为在 React 的项目中会用到 ES6(即ES2015) 的语法，而 ES6 还没有得到广泛的支持，所以我们需要借用 Babel 这个工具将我们使用了 ES6 语法的代码转换成 ES5 的语法，从而可以在更广泛的环境下运行。

    怎么安装 Babel？

    安装 babel-loader babel-core babel-preset-es2015
    现在只要添加一个 .babelrc 文件即可，babelrc 是 Babel 的配置文件
    只需在 .babelrc 中写入如下内容

    • { “presets”: [“es2015”, “stage-0”] }

    将 webpack.config.js 文件做如下的修改：

    module: {
  loaders: [{
    test: /\.js$/,
    loaders: ['babel'],
    include: path.join(__dirname, 'src')
  }]
},
devServer: {
  stats: 'errors-only'
}
