react 环境搭建

react环境

1.安装node.js 和 npm （node中包含npm）

nvm install node

2.安装brew

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

3.安装nvm [http://yijiebuyi.com/blog/b1328ffe88cdde6b4102894635cf8f11.html]

curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.32.1/install.sh | bash

4. 安装 zsh

curl -L https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh | sh

5. Git

Mac默认安装git.

6. 安装flow

brew install flow

7. 安装eslint

编辑器：atom

atom 插件：

  1. file-icon 文件图标
  2. Git-status git状态
  3. linter
  4. nuclide
  5. terminal-plus 终端/命令行工具
  6. wakatime
  7. autocomplete-plus 当输入的时候，提供可能的候选项
  8. markdown-preview && markdown-scroll-sync 将markdown-preview编辑区和预览区同步滚动
  9. docblockr 用于写注释的插件

react调试插件：
